a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git fue diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.
b. ¿A quién pertenece actualmente Github y por qué?
Github pertenece acutalmente a Microsoft con el objetivo principal de que el mundo esté preparado para los nuevos cambios que se avecinan.
