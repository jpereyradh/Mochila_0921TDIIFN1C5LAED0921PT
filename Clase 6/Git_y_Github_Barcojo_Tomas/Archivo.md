## CHEATSHEET <img src="https://media.giphy.com/media/hvRJCLFzcasrR4ia7z/giphy.gif" width="25px"></a>

Clonar un repo bien cheto
```bash
git clone
```

Crear nuevo repo
```bash
git init
```

Pushear al repo
```bash
git push
```

Commitear cambios y nuevos archivos
```bash
git commit -m
```

Añadir cambios y nuevos archivos
```bash
git add .
```
```bash
git add 'archivo'
```
